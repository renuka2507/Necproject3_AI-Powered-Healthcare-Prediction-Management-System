# 🏥 AI-Powered Healthcare Prediction & Resource Management System

## Project Overview

The AI-Powered Healthcare Prediction & Resource Management System is an intelligent healthcare platform that helps hospitals predict patient outcomes, improve treatment planning, and optimize hospital resources efficiently.

## 🚀 Installation

```bash
git clone https://github.com/renuka2507/Necproject3_AI-Powered-Healthcare-Prediction-Management-System.git
cd Necproject3_AI-Powered-Healthcare-Prediction-Management-System
pip install -r requirements.txt
streamlit run app.py
```
